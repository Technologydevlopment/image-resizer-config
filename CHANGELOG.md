# Changelog

All notable changes to this project will be documented in this file.

## [1.0.5] - 2026-03-27

### Added
- **Premium Update UI**: Redesigned update verification with a glassmorphic centered dialog.
- **Sync Visuals**: Real-time 'Cloud Sync' loading overlay for the Home Screen.
- **UX Refinement**: Automatic drawer closure for seamless update verification.

## [1.0.4] - 2026-03-27

### Added
- **Cloud Config System**: Decentralized update system using a public GitHub JSON configuration.
- **Mandatory Updates**: Forced "App Lock" screen for critical security patches.
- **Self-Update UX**: Integrated manual 'Check for Updates' in the App Drawer.

## [1.0.3] - 2026-03-25

### Added
- **Legal Compliance**: Comprehensive audit of Privacy Policy and Terms of Use.
- **In-App Screens**: Dedicated `PrivacyPolicyScreen` and `TermsOfUseScreen`.
- **Root Documentation**: Added `SECURITY.md`, `TERMS_OF_USE.md`, and refined `README.md`.
- **Navigation**: Integrated legal links directly into the Settings screen.

### Fixed
- **Banner Ads**: Optimized ad space allocation in Home, Editor, Crop, and Fit screens.
- **Theming**: Integrated `BannerPlaceholder` with dark/light theme support.
- **Security**: Formalized "Local-First" architecture commitment (zero data collection).

### Changed
- **Branding**: Finalized "Industrial Pro" Amber and Black design system consistency.
- **Support**: Updated support channels to `jasu1070@gmail.com`.

## [1.0.2] - 2026-03-24

### Added
- **Splash Screen**: Professional, procedural splash screen.
- **Editor Tools**: High-fidelity "Resize" and "Fit" toolbars.
- **Performance**: Integrated high-performance C++ (NDK) image engine.

### Fixed
- **UI Layout**: Resolved overlap and safe area issues on Android devices.
- **Navigation**: Stabilized drawer and bottom panel constraints.
